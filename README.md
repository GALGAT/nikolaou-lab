#Nikolaou Research Group Website
This repository contains the source code for the official website of Dr. Michael Nikolaou's Research Group at the University of Houston. The group focuses on the intersection of computer-aided process engineering, mathematical modeling, and data science.

To run this site locally on your machine:

Clone the repository:

Bash
git clone https://github.com/hgalgat/nikolaou-lab-site.git
Install Jekyll and Bundler:

Bash
gem install jekyll bundler
Launch the local server:

Bash
bundle exec jekyll serve
View the site: Open http://localhost:4000 in your browser.
